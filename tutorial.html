How to Run Our Bypasses

Step-by-Step Instructions

  1. Unzip the .zip file you received and place the folder on your desktop. (It must be on the desktop or it won't work.)
  2. Ensure that SIP (System Integrity Protection) is disabled. The script will not function otherwise.
    3. SIP needs to be disabled because Apple prevents 3rd party software from injecting. You can re-enable it after you are done.
  3. If you're using an M Chip (Apple Silicon) Mac, follow this tutorial:
    Watch SIP Disable Tutorial for M1/M2 Macs
  4. For Intel Mac users:
    • Shut down your laptop completely.
    • Press the power button, and when you hear the chime, hold CMD + R to enter recovery mode.
    • Continue following the steps from the tutorial above.
  5. After SIP is disabled, open the unzipped folder.
  6. Right-click on the Penguin app > choose Open.
  7. Once opened, run the exam software directly.
  8. If you see an "unidentified developer" error:
    • Go to Settings > Privacy & Security
    • At the bottom, click "Allow Anyway"
